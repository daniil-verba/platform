## 🔥 Что такое Platform?

**Platform** — это мессенджер, работающий поверх децентрализованной P2P-сети **Hivemind**.

- 🔒 Сквозное шифрование всех сообщений.
- 🌐 Нет центральных серверов — только P2P.
- 🇷🇺 Российская разработка, импортозамещение.
- 🧠 Каждый новый пользователь усиливает сеть.

## 🔧 Сборка из исходников

### Требования
- CMake 3.15+
- Компилятор с C++20
- Hivemind (устанавливается отдельно или подключается как submodule)

### Сборка
```bash
git clone https://github.com/daniil-verba/platform-app.git
cd platform-app
mkdir build && cd build
cmake ..
make -j$(nproc)
./platform
```
