# Platform

> **Официальный клиент сети Hivemind.**
>
> Первое приложение нового поколения — работает поверх единой децентрализованной P2P-сети.

---

## 🔥 Что такое Platform?

**Platform** — это не просто мессенджер. Это **первый клиент сети Hivemind** — децентрализованной P2P-сети, где каждый пользователь одновременно является и клиентом, и частью инфраструктуры.

- 🔒 **Сквозное шифрование** — никто, кроме отправителя и получателя, не может прочитать сообщение.
- 🌐 **Нет центральных серверов** — сообщения идут напрямую от пользователя к пользователю.
- 🇷🇺 **Российская разработка** — создано в России, для России, с открытым исходным кодом.
- 🧠 **Каждый новый пользователь усиливает сеть** — чем больше людей, тем надёжнее и быстрее работает связь.

## 🌐 Сеть Hivemind

Platform работает поверх [Hivemind](https://github.com/daniil-verba/hivemind) — единой децентрализованной P2P-сети.

```
┌─────────────────────────────────────────────────────────────┐
│                      HIVEMIND NETWORK                        │
│           Единая децентрализованная P2P-сеть                │
│    Каждый узел = хранилище + маршрутизатор + ретранслятор   │
└─────────────────────────────────────────────────────────────┘
                              │
            ┌─────────────────┼─────────────────┐
            │                 │                 │
            ▼                 ▼                 ▼
       ┌─────────┐      ┌─────────┐      ┌─────────┐
       │Platform │      │RPG Game │      │Website  │
       │(чат)    │      │(чат)    │      │(поддержка)│
       └─────────┘      └─────────┘      └─────────┘
            │                 │                 │
            └─────────────────┴─────────────────┘
                              │
                    Все используют ОДНУ сеть
            Больше приложений → сильнее сеть → лучше всем
```

**Platform растёт → Hivemind растёт → любое приложение на базе Hivemind получает готовую аудиторию.**

## 🔧 Сборка из исходников

### Требования

- CMake 3.15+
- Компилятор с C++20
- Hivemind (устанавливается отдельно или подключается как submodule)

### Сборка

```bash
git clone https://github.com/daniil-verba/platform.git
cd platform
mkdir build && cd build
cmake ..
make -j$(nproc)
./platform
```

## 🎯 Для разработчиков

Platform — это **первый клиент**, но не единственный. Hivemind открыт для любых приложений:

- **Игры** — встроенный чат, лидерборды, мультиплеер без серверов.
- **Сайты** — чат поддержки, реалтайм-уведомления, динамические страницы.
- **Приложения** — синхронизация данных, обмен сообщениями.

Все приложения работают в **одной сети**. Пользователи Platform автоматически становятся узлами для вашего приложения.
